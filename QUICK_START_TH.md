# PDF Reader - Quick Start

## 🚀 สำหรับผู้ใช้งานภาษาไทย

### ติดตั้งและใช้งานอย่างรวดเร็ว

#### ขั้นตอนที่ 1: ติดตั้ง Dependencies

**สำหรับ Windows:**

1. ติดตั้ง **Visual Studio 2022** (Community Edition ฟรี)
   - ดาวน์โหลดจาก: https://visualstudio.microsoft.com/
   - เลือก "Desktop development with C++"

2. ติดตั้ง **vcpkg** (ตัวจัดการแพ็คเกจ)
   ```powershell
   git clone https://github.com/Microsoft/vcpkg.git
   cd vcpkg
   .\bootstrap-vcpkg.bat
   ```

3. รันสคริปต์ติดตั้ง dependencies
   ```powershell
   cd e:\dev\pdf_reader
   .\setup_dependencies.bat
   ```

#### ขั้นตอนที่ 2: Build โปรแกรม

```powershell
.\build.bat
```

#### ขั้นตอนที่ 3: รันโปรแกรม

```powershell
# รันโปรแกรมเปล่า
.\run.bat

# หรือเปิดไฟล์ PDF เลย
.\run.bat "path\to\your\document.pdf"
```

## ⌨️ คีย์ลัดที่ควรรู้

### การใช้งานพื้นฐาน
- `Ctrl+O` - เปิดไฟล์
- `Ctrl+P` - พิมพ์
- `Ctrl+Q` - ปิดโปรแกรม
- `F11` - โหมดเต็มจอ

### การซูม
- `Ctrl++` - ซูมเข้า
- `Ctrl+-` - ซูมออก
- `Ctrl+0` - ขนาดจริง (100%)
- `Ctrl+เมาส์วีล` - ซูมด้วยเมาส์

### การเปลี่ยนหน้า
- `Page Down` หรือ `→` - หน้าถัดไป
- `Page Up` หรือ `←` - หน้าก่อนหน้า
- `Home` - หน้าแรก
- `End` - หน้าสุดท้าย

### การหมุน
- `Ctrl+L` - หมุนซ้าย
- `Ctrl+R` - หมุนขวา

### การค้นหา
- `Ctrl+F` - เปิดหน้าต่างค้นหา

## 🎯 วิธีใช้งานเบื้องต้น

### เปิดไฟล์ PDF

**วิธีที่ 1:** คลิก `File` → `Open...` แล้วเลือกไฟล์

**วิธีที่ 2:** กด `Ctrl+O` แล้วเลือกไฟล์

**วิธีที่ 3:** รันจาก command line
```powershell
.\run.bat "C:\Documents\myfile.pdf"
```

### ซูมเข้า/ออก

**ด้วยเมาส์:**
- กด `Ctrl` ค้างแล้วเลื่อนวงล้อเมาส์

**ด้วยแถบเครื่องมือ:**
- คลิกปุ่ม `+` หรือ `-`
- ลากแถบเลื่อนซูม

**ด้วยคีย์บอร์ด:**
- `Ctrl++` ซูมเข้า
- `Ctrl+-` ซูมออก

### เปลี่ยนหน้า

**ด้วยคีย์บอร์ด:**
- กด `Page Down` หรือลูกศรขวา

**ด้วยเมาส์:**
- คลิกปุ่ม ◀ หรือ ▶
- พิมพ์เลขหน้าในช่อง
- คลิกภาพย่อด้านซ้าย

### ค้นหาข้อความ

1. กด `Ctrl+F`
2. พิมพ์คำที่ต้องการค้นหา
3. กด Enter หรือคลิก "Search"
4. คลิกผลลัพธ์เพื่อไปที่หน้านั้น

### พิมพ์เอกสาร

1. กด `Ctrl+P`
2. เลือกเครื่องพิมพ์
3. ตั้งค่าการพิมพ์
4. คลิก "Print"

## ❓ แก้ปัญหาเบื้องต้น

### ปัญหา: Build ไม่ผ่าน

**แก้ไข:**
1. ตรวจสอบว่าติดตั้ง Visual Studio 2022 แล้ว
2. ตรวจสอบว่า vcpkg ติดตั้ง dependencies ครบแล้ว
3. รัน `setup_dependencies.bat` อีกครั้ง

### ปัญหา: เปิด PDF ไม่ได้

**แก้ไข:**
1. ตรวจสอบว่าไฟล์เป็น PDF จริง
2. ลองเปิดด้วยโปรแกรมอื่นก่อน
3. ตรวจสอบว่าไฟล์ไม่เสียหาย

### ปัญหา: ตัวอักษรเบลอ

**แก้ไข:**
1. เพิ่มระดับซูม
2. กด `Ctrl+0` เพื่อรีเซ็ตเป็นขนาดจริง
3. ตรวจสอบว่า PDF มีความละเอียดพอ

### ปัญหา: ค้นหาไม่เจอข้อความ

**แก้ไข:**
1. PDF อาจเป็นรูปภาพ (ไม่มี text layer)
2. ลองใช้คำค้นหาอื่น
3. ตรวจสอบการสะกด
